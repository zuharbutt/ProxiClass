📋 How to Find Your Bluetooth MAC Address
📱 Android:
Settings → About Phone → Status → Bluetooth Address
🍎 iPhone:
Settings → General → About → Bluetooth (shown as Wi-Fi Address on some models)
💻 Windows Laptop:
Settings → Bluetooth & devices → More Bluetooth settings → Hardware tab → Address
⚠️ Format: Enter exactly as shown, e.g. AA:BB:CC:DD:EE:FF. The teacher's scanner matches this address.
